Voici l'instruction principale pour ce programme


make        -> pour compiler le programme

make chmod -> pour donner les droits de creation d'autres au fichier vector2tex.sh
make latex -> pour installer latex si cest pas le cas alors on peut le sauter

make clean  -> pour nettoyer
./bin/main  -> pour executer le programme 

et puis suivre les instructions
