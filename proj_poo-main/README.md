<div align="center">
  <h1 style="font-size: 32px; font-family: 'Times New Roman', serif;">Instruction et Utilisation</h1>
</div>

Pour compiler le programme:
```bash
make		
```

Pour donner les droits de creation d'autres au fichier vector2tex.sh:
```bash
make chmod 	
```

Pour installer latex si cest pas le cas alors on peut le sauter:
```bash
make latex 	
```

Pour nettoyer:
```bash
make clean  
```

Pour executer le programme:
```bash
./bin/main  
```

Apres suivez les instructions affiches
