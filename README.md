# Kakuro Solver 🧩

Ce projet est un solveur de grilles **Kakuro** développé dans le cadre 
d'une architecture **orientée objet (POO)**.

## Fonctionnalités

- Résolution automatique d'une grille Kakuro
- Plusieurs modes de sortie au choix :
  - Affichage dans le **terminal**
  - Écriture dans un **fichier texte**
  - Génération d'un **fichier PDF** via un script LaTeX 
    (génère d'abord un `.tex`, puis le compile en `.pdf`)
- Lecture de grilles depuis un **fichier** ou saisie **manuelle**

## Architecture

Le projet est entièrement construit selon les principes de la 
**Programmation Orientée Objet (POO)**.

Pour une description détaillée de l'architecture, des diagrammes 
de classes et des exemples visuels, veuillez consulter le fichier PDF 
inclus dans ce dépôt.
